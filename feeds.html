CVE-2025-9626

Published: 2025-10-11 05:15:44 CDT

Severity: MEDIUM

CVSS Score: 4.3

The Page Blocks plugin for WordPress is vulnerable to Cross-Site Request Forgery in all versions up to, and including, 1.1.0.
