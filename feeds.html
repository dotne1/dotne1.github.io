Published: 2026-01-14 00:15:55 CDT

Severity: MEDIUM

CVSS Score: 6.4

The SearchWiz plugin for WordPress is vulnerable to Stored Cross-Site Scripting via post titles in search results in all versions up to, and including, 1.0.0.
